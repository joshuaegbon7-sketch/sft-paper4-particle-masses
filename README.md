# Spacefield Transformation IV: Particle Masses from Primitive Closure Invariants

## Overview

This repository accompanies:

**Joshua Egbon, Spacefield Transformation IV: Particle Masses from Primitive Closure Invariants.**

The repository provides reproducible calculations for the particle-mass constructions derived in the paper.

The framework derives charged-lepton, quark, neutrino, and hidden-sector mass scales from the primitive closure kernel

\[
\mathcal K=\operatorname{diag}(2,3,4).
\]

No particle-specific mass parameters are introduced in the leading-order construction.


## Primitive Closure Invariants

The particle-mass framework is generated from the primitive closure invariants

\[
\chi_M=2,
\qquad
\chi_A=3,
\qquad
\chi_B=4.
\]

These induce the primitive closure kernel

\[
\mathcal K=\operatorname{diag}(2,3,4).
\]
